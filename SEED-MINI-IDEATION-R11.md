# Seed Mini Ideation Round 11

## Film Score (NEW PRODUCT VERTICAL)
SceneToMusic: scene description → terrain + mode + parameters
"Night, lonely highway, driving rain" → sparse minor, slow, low register
API: analyze_scene(description) → SceneConfig(terrain, mode, key, bpm, texture)
Could generate entire film scores from screenplay analysis.

## Constraint Puzzles (EDUCATION)
Musical Sudoku: voice-time grid, some notes filled in, complete it following rules.
- No two adjacent same notes
- Must resolve leading tones
- Each voice must span ≥ octave
Generator scales from easy (2 voices, 4 bars) to insane (12 voices, 32 bars)
Educational gold: learn constraint theory by solving puzzles.

## Duet Protocol (MULTI-AGENT)
Priority-based mediation: melody leads, harmony adapts.
Shared consensus layer. Hard constraint enforcement prevents clashing.
3 methods: negotiate(), propose_counterpoint(), resolve_conflict()
→ Multi-agent AI jam sessions.

## Year in Review (VIRAL FEATURE)
5 cards: total hours, best genre, longest flow, most improved order, unexpected discovery.
Shareable carousel. Like Spotify Wrapped but for practice.
Built from flow tracker data already collected.
