### Key Patterns Discovered
1. **Branch mismatch**: Many repos have default_branch=master but CI triggers on main
2. **Workflow bloat**: Agent-generated repos have 5-6 workflows (backup-recovery, security-scan, dependency-updates, website-deploy) that total 1000+ lines
3. **Integer division bugs**: `checked_div(50, 100)` returns 0 in Rust, not 0.5
4. **npm ci failures**: Many repos have no package-lock.json, need `npm install` instead
