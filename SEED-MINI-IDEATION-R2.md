# Seed Mini Ideation Round 2

## TARAB (Arabic طرب)
"Joyful rapture" — the exact instant a microtonal pivot note in a maqam sends physical shivers through listeners, suspending time. 
- This is the HOLY GRAIL state — beyond duende (dark fire), tarab is the listener's experience of duende
- Add to vocabulary: TARAB = the audience's flow state (duende = performer's flow state)

## CODE-AS-MUSIC

## 5 TESTABLE FLOW HYPOTHESES
1. Skill-based stratification: pros hit flow faster, stay longer
2. Peak flow: 20-40 minutes into practice, not at start
3. Flow breakers: switching tasks, unexpected errors, external interruption
4. Flow builders: familiar warmup → progressive challenge → peak
5. Cross-genre transfer: flow in jazz practice predicts flow in production
